PHP Kai Harness

Pluggable AI Agent Harness Framework for PHP & Laravel

Overview

PHP Kai Harness (k415mm/phpkaiharness) is a robust, modular Object-Oriented Programming (OOP) framework for building autonomous AI agent workflows in PHP 8.2+. It provides a step-by-step agent execution loop, dynamic tool registries, quantum/cognitive vector memory, LLM client adapters (Ollama, Qwen, OpenRouter, LM Studio, Laravel AI SDK), context compaction, and optional full Laravel integration.

Key Features

  • Autonomous Agent Loop: Coordinates multi-step reasoning, tool execution, and response synthesis.
  • Dynamic Tool Registry: Register local PHP callbacks or remote microservice tools with JSON schema definitions.
  • Vector & Cognitive Memory: Retains context across sessions with SQLite vector search and semantic memory graph.
  • LLM Provider Pipeline: Built-in rate limiting, provider failover, prompt optimization, and thinking budget controls.
  • Laravel Integration: Includes ServiceProvider auto-discovery, published config, Artisan commands, and a Blade telemetry dashboard.
