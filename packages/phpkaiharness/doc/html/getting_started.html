1. Installation

Install via Composer in your PHP or Laravel project:

composer require K415mm/phpkaiharness

2. Basic Standalone PHP Usage

You can run agents in standard PHP scripts without framework overhead:

use Phpkaiharness\Core\AgentHarness;
use Phpkaiharness\Llm\LaravelAiClient;

$client = new LaravelAiClient('ollama', 'llama3.2');
$harness = new AgentHarness($client, maxIterations: 5);

$result = $harness->run(
    systemPrompt: "You are a helpful coding assistant.",
    userPrompt: "Write a PHP function to check if a string is a palindrome."
);

echo $result['response'];

3. Laravel Framework Setup

In Laravel, the package automatically registers its PhpkaiharnessServiceProvider via Composer package discovery.

Publish configuration options and telemetry views:

php artisan vendor:publish --tag=phpkaiharness-config
php artisan vendor:publish --tag=phpkaiharness-views

Access the telemetry dashboard at: /harness/dashboard
